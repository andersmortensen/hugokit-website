# {{ site.Title }}

> HugoKit is a free Mac app for the Hugo static site generator: it runs the local server with live-reloading preview, edits content and config, scores a site's health, checks every site before publishing (preflight with approve-a-diff auto-fixes), and publishes to GitHub Pages or your own server over FTP/SFTP. No terminal needed. It works as a companion to any code editor or AI coding agent – content stays plain Markdown on disk, and the preview reloads on save.

- Free: no license, no subscription, no account, no cloud
- Requires macOS 26 (Tahoe) or later
- Personal, non-commercial project by Anders Mortensen (Denmark)
