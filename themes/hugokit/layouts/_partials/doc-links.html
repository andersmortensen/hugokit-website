{{- /* Docs-links til en feature. Kontekst: en slice af docs-stier fra _index.md, evt. med #anker: "/docs/getting-started/#create-a-new-site". Linket er en chip, ikke en tekstlinje: dokument-ikon + "Docs" som kicker + sidens egen titel + pil – så det er tydeligt at man forlader forsiden og lander i den tekniske beskrivelse. Labelen hentes fra dokumentationen selv – afsnittets overskrift når der er et anker, ellers sidens LinkTitle – så teksten aldrig kan drifte fra docs. Findes siden (eller ankeret) ikke, renderes intet: et forkert link er værre end intet link. */ -}} {{- range . -}} {{- $parts := split . "#" -}} {{- $path := index $parts 0 -}} {{- $anchor := "" -}} {{- if gt (len $parts) 1 }}{{ $anchor = index $parts 1 }}{{ end -}} {{- $p := site.GetPage $path -}} {{- with $p -}} {{- $href := $p.RelPermalink -}} {{- $label := $p.LinkTitle -}} {{- if $anchor -}} {{- $href = printf "%s#%s" $p.RelPermalink $anchor -}} {{- /* .Fragments.Headings har en syntetisk rod på niveau 0 – sidens h2'er ligger som dens børn (samme mønster som docs-nav). */ -}} {{- $hs := $p.Fragments.Headings -}} {{- if and $hs (eq (index $hs 0).Level 0) }}{{ $hs = (index $hs 0).Headings }}{{ end -}} {{- range $hs }} {{- if eq .ID $anchor }}{{ $label = .Title | plainify | htmlUnescape }}{{ end -}} {{- /* Ankeret kan være en h3 under en h2 (fx "Hugo" i First launch) */ -}} {{- range .Headings }} {{- if eq .ID $anchor }}{{ $label = .Title | plainify | htmlUnescape }}{{ end -}} {{- end -}} {{- end -}} {{- end -}} Docs · {{ $label }} → {{- end -}} {{- end -}}
