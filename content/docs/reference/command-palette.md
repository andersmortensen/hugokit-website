The command palette is HugoKit's universal action launcher. Press ⌘K from
anywhere in the app to open it.

## What it searches

- Built-in actions: start/stop server, open config, new content, publish, etc.
- Site switches — typing a site name jumps to it
- Settings — any settings page is searchable by name
- Recent content — typing part of a post title jumps to editing it

## Keyboard

- **↑ / ↓** — navigate results
- **Enter** — execute selected action
- **Esc** — close the palette without action
- **Tab** — cycle between result categories
