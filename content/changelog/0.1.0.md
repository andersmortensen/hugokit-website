The first release of HugoKit.

## New

- Every Hugo site in one window — add sites one by one, or point HugoKit at a folder and it finds them itself
- One-click preview: start and stop Hugo's server, with toggles for drafts, future and expired content
- Live logs translated into plain language, and a monospace preview of the exact command before it runs
- Publish to GitHub Pages and your own server over FTP/SFTP — one target or several, with one "Publish all"
- Preflight before anything ships: broken links, subpath traps, config and asset issues — every fix shown as a diff first
- Create new sites from the HugoKit Starter template or a blank Hugo site, git init included
- Theme Gallery: browse the official Hugo themes with screenshots, and install one with a click
- Template preview (⇧⌘T): every layout file in your theme, rendered live
- Content overview: pages, sections, word counts and a front matter health check per site
- Config editor — visual or raw TOML, with a diff preview before anything is saved
- Hugo Reference: searchable Hugo documentation built into the app
- Command palette (⌘K), menu bar extra, and per-site emoji
- HugoKit finds your Hugo install automatically — and if there isn't one, it installs it for you

## Notes

- Free — no account, no cloud; deploy credentials live in the macOS Keychain
- Requires macOS 26 (Tahoe) or later
