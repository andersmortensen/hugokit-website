The problem

Hugo is wonderful. The black window it lives in is not.

HugoKit runs the commands, reads the logs, and turns the whole thing into buttons you can actually find. The terminal is still there — you just never have to look at it.

What it does

Everything between new site and live.

Every site in one window

Add your sites one by one — or point HugoKit at a folder and it finds them itself.

One-click preview

Start and stop the server with a click. Drafts included, logs translated into plain language.

Publish anywhere

GitHub Pages or your own host over SFTP. Or both at once, with one “Publish all”.

Preflight auto-fix

Broken links and subpath traps are caught before they ship — and fixed for you, diff first.

Theme Gallery

Browse the official theme gallery and install one with a click. No git clone required.

Honestly native

SwiftUI, menu bar, Keychain. No Electron, no account — nothing ever leaves your Mac.
