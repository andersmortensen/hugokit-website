A focused set of tools for managing Hugo sites — designed to get out of your way.

Multi-site management

All your Hugo sites in one sidebar. Isolated servers, configs, and deploys — switch context in a single click.

One-click server

Start, stop, restart with live logs. Drafts, ports, and flags as simple switches — no terminal needed.

Deploy to GitHub Pages

One-click publish. OAuth, Actions workflow, preflight validation — the entire pipeline automated.

Easy config editor

Edit hugo.toml with a form view or raw text. See a diff before saving — never break your site accidentally.

Auto-detect Hugo

Finds your Hugo binary on launch — or installs it for you. Zero config.

Theme gallery

Browse, preview, and install Hugo themes from themes.gohugo.io — without leaving the app.

Content overview

Page counts, word counts, drafts, and front matter health at a glance.

Built-in Hugo reference

Searchable docs across 9 categories. Pin your most-used references.

Control from menu bar

Live server status and quick controls without opening the main window.

Full shortcut coverage

Works with any editor

Native notifications

What is Hugo?

Hugo is one of the world's most popular open-source static site generators.

HugoKit is a native macOS app that wraps Hugo in a friendly interface — so you can manage sites, preview themes, and publish without ever opening Terminal.
